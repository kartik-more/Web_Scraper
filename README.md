# 🕸️ Full Stack Web Scraper (Python + HTML, CSS, JS)

A robust and scalable **full-stack web scraping project** that collects product data from an e-commerce website using **Python (backend scraping engine)** and presents it through an interactive **frontend built with HTML, CSS, and JavaScript**.

This project demonstrates real-world concepts like **data extraction, processing, storage, and visualization**, making it ideal for learning and portfolio showcase.

---

## 📖 Project Overview

In today’s data-driven world, extracting meaningful information from websites is an essential skill. This project simulates a real-world use case where product data is collected, structured, and displayed in a user-friendly interface.

The system follows a simple pipeline:

> 🌐 Website → 🐍 Python Scraper → 📊 CSV Data → 💻 Frontend UI

---

## ✨ Key Features

### 🔹 Data Extraction
- Scrapes real-time product data from an online e-commerce website
- Extracts:
  - 📌 Product Name
  - 💲 Price
  - ⭐ Rating

### 🔹 Data Processing
- Cleans and organizes raw HTML data
- Converts unstructured data into structured format

### 🔹 Data Storage
- Stores extracted data into a **CSV file**
- Ensures easy portability and analysis

### 🔹 Frontend Visualization
- Displays data in a clean and responsive UI
- Built using pure HTML, CSS, and JavaScript
- Can be extended with search, filters, and sorting

### 🔹 Modular Design
- Separation of concerns:
  - Backend (scraper)
  - Frontend (UI)
- Easy to maintain and extend

---

## 🛠️ Tech Stack

### 🚀 Backend (Data Scraping)
- **Python** – Core programming language
- **Requests** – Handles HTTP requests
- **BeautifulSoup (bs4)** – Parses HTML content
- **Pandas** – Data manipulation and CSV export

### 🎨 Frontend (User Interface)
- **HTML** – Structure of web pages
- **CSS** – Styling and layout
- **JavaScript** – Dynamic data rendering
